## 🛠️ Tech Stack & Architecture

* **Framework:** **Next.js 14** (App Router) for Server-Side Rendering (SSR) and optimized performance.
* **Styling:** **Tailwind CSS** using a custom **Zinc-950** and **Cyan-400** color palette.
* **Content:** **Sanity CMS** (Headless) for real-time content management.
* **Animations:** **Framer Motion** for high-end scroll reveals and transitions.
* **Deployment:** **Vercel** for seamless CI/CD.

---

## ✨ Key Features

* **Real-time CMS Integration:** Content updates reflect instantly via revalidation.
* **Responsive Architecture:** Fully optimized for mobile, tablet, and desktop.
* **Interactive UI:** Animated project cards and custom navigation.
* **Performance First:** Optimized images and near-perfect Lighthouse scores.
